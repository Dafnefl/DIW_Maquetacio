# Homies & Burgers — Pràctica 10 (Disseny Responsive)

## 📌 Descripció del projecte

Aquesta pràctica amplia el projecte anterior incorporant **disseny responsive** mitjançant media queries.

L’objectiu és adaptar la interfície a diferents dispositius sense modificar l’estructura HTML original.

---

## 🎯 Objectius

- Aplicar disseny responsive amb CSS.
- Adaptar el layout a mòbils i tablets.
- Utilitzar herència i evitar duplicació de codi CSS.
- Mantenir compatibilitat entre navegadors.
- Garantir accessibilitat.
- Treballar amb branca específica de Git.

---

## 🌐 Breakpoints utilitzats

### 📱 Mòbil (≤ 576px)

- Menú en format vertical
- Botons a amplada completa
- Secció “Sobre nosaltres” en una sola columna
- Carta simplificada
- Galeria en columna
- Footer centrat i sense logotip

---

### 📟 Tablet (≤ 768px)

- Menú simplificat i centrat
- Reducció de complexitat en Grid
- Galeria flexible amb ajust automàtic
- Footer centrat i amb menys elements

---

## 🧱 Millores implementades

### ♻️ Optimització CSS

- Ús d’herència per evitar repeticions
- Media queries netes i modulars
- Reutilització de classes existents

### 📐 Adaptació de layout

- Flexbox i Grid es mantenen
- Només es modifiquen comportaments
- Espaiat i alineació coherents

### 📱 Experiència mòbil

- Interfície simplificada
- Botons accessibles
- Textos millor llegibles
- Imatges escalables

---

## 🛠️ Tecnologies utilitzades

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Variables CSS
- Disseny responsive

---

## ♿ Accessibilitat

- Estructura HTML semàntica
- Imatges amb atribut `alt`
- Compatibilitat amb WAVE
- Navegació clara

---

## 🌍 Compatibilitat

Testat en:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

---

## 📱 Estratègia responsive

- Adaptació progressiva per dispositius
- Sense modificació HTML
- Només CSS (requisit obligatori)
- Prioritat a la reutilització de codi

---

## ⚠️ Restriccions complertes

- Només modificació CSS
- Sense duplicació de codi en media queries
- Sense scroll horitzontal
- Estructura original respectada
- Codi net i comentat
